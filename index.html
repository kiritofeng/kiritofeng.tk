School & Projects
Sorting Visualizer

An assignment I was given in my ICS4U class was to visually compare two sorting algorithms. As the class was in Java, my partner and I decided to compare Quicksort and Timsort.

The project was implemented using javax.swing for the GUI, and scales seemlessly as the window is resized. There is also an option to change the number of bars, and also a worst case option for both sorts.

Projects
P3 Pointer

The P3 Pointer was the first project I built at a hackathon. It used a 9 degrees of freedom sensor and vector calculations to act as a wearable pointing device, sending the data via an ATmega32U4.

As the first project I built at a hackathon, and the first hardware project I'd ever done, it was a great learning experience for me.
